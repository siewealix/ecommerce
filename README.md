# Ecommerce web (React)

Point de départ simple pour une version web en React, inspirée de la structure
épurée du projet mobile React Native. Le but est de progresser étape par étape
avec des composants faciles à lire et à modifier.

## Démarrer le projet

1. Installez les dépendances :

   ```bash
   npm install
   ```

2. Lancez le mode développement :

   ```bash
   npm run dev
   ```

3. Ouvrez l'URL affichée dans le terminal (par défaut http://localhost:5173).

Le coeur de l'interface se trouve dans `src/App.jsx` : modifiez ce fichier pour
ajouter de nouvelles sections (navigation, panier, compte utilisateur, etc.).
Le style de base est défini dans `src/index.css`.
